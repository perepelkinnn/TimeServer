Сервер "точного" времени на udp сокетах
Запускать от имени администратора
Серверу посылается ntp пакет, он разбирает пакет, заполняет нужные поля, запаковывает и отправляет обратно.
Для проверки можно запустить request.py который подключается к серверу отпраляет составляет пакет и отправляет на сервер, затем получает от сервера пакет, разбирает его и выводит поля на экран
По умолчанию сервер и клиен работают по адресу localhost. Server port - 11000, client port 11001
Чтобы изменить сдвиг времени достаточно поменять число в файле config.txt
Осуществляется логирование в файл requests.log
