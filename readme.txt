Сервер "точного" времени на udp сокетах
Запускать от имени администратора
Серверу можно послать любое сообщение на что он ответит "точным" временем
Для проверки можно запустить request.py который подключается к серверу отпраляет слово "request", получает от сервера время и выводит его на экран
По умолчанию сервер и клиен работают по адресу localhost
Чтобы изменить сдвиг времени достаточно поменять число в файле config.txt
Осуществляется логирование в файл requests.log
